# Projeto: Desenvolvimento de uma Solução de Análise de dados da BODIVA, para apoiar os investidores na tomada de decisão.

Dashboard de Business Intelligence desenvolvido para análise do mercado de capitais angolano, com foco nos dados da BODIVA (Bolsa de Dívida e Valores de Angola).

O projeto permite acompanhar indicadores do mercado de ações e títulos de tesouro através de visualizações interativas.


## Funcionalidades

### Mercado de Ações

- Montante negociado
- Empresas cotadas
- Quantidade de negócios
- Ranking de ações por preço
- Volume financeiro negociado por empresa


### Mercado de Títulos de Tesouro

- Monitorização dos ativos negociados
- Preço dos títulos
- Montante movimentado
- Títulos mais negociados
- Análise de liquidez


## Tecnologias Utilizadas

- Power BI
- DAX
- Power Query
- SQL
- PostgreSQL
- HTML/CSS/SVG
